# transfinite-side-nesting
無限回貫通問題は存在しない

## 集合
### 前提
0. 0は自然数に含むものとする。
1. 自然数全体の集合をNとする。
1. Nに文字ωを追加した集合をN'とする。
1. Σを(と)と零と+とN'の元からなる集合とする。Σの元を'''トークン'''と呼ぶ。
1. Σの元0個以上からなる配列全体の集合をΣ*と書く。

### 配列
1. A,B∈Σ*に対し、A結BでAとBの連結を表す。

### 項
Σ*の部分集合TとPTを以下で定める。
1. 零∈Tである。
2. いかなる(a,b)∈PT×(T\{0})に対しても、A結\[+\]結B∈Tである。
3. いかなる(m,n,t)∈N'×N'×Tに対しても、\[(,m,n\]結t結\[)\]∈T∩PTである。

## 略記
1. Tの元に対し、零を0と略記し、A結\[+\]結BをA+Bと略記し、\[(,m,n\]結t結\[)\]をψ^m_n(t)またはψ_n^m(t)と略記する。
  - Tの元では零と0は出現する箇所が異なるため混乱を生むことはない。

Tの元に対する略記を以下のように追加する。
1. $0=0とする。
2. $1=ψ^0_0(0)とする。
3. 2以上の自然数nに対し、$n=$(n-1)+$1とする。
4. $ω=ψ^0_0($1)とする。
